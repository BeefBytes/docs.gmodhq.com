# GmodHQ Docs
Documentation and rules for GmodHQ network

## Usage

Clone repo<br />
```
git clone git@github.com:BeefBytes/docs.gmodhq.com.git
```

Start development server<br />
```
docker-compose up watch
```

Build docs<br />
```
docker-compose up build
```
